## 目標用戶輪廓

- 職稱：行銷經理
- 公司規模：百大企業
- 日常工作：週報/月報撰寫、戰報分析、廠商開發與聯繫、新產品企劃、出差報帳
- 常用工具：Email、LINE、Excel、Google Drive、PowerPoint
- 主要痛點：重複性報告耗時、資料散落各處難整合、廠商溝通量大

---

## 五大核心功能模組

### 1. 📝 報告生成器
- 週報 / 月報一鍵生成
- 語音輸入轉文字加入素材庫
- 上傳過去報告讓 AI 學習格式
- 自動填入數據（YoY 比較）
- 輸出格式：Word / PDF / PPT

### 2. 📊 數據整理工具
- Excel / CSV 戰報分析整合
- 自動生成比較圖表
- 去年同期 / 本月數據對比
- 關鍵字績效追蹤
- 雲端資料夾同步

### 3. ✉️ 溝通管理中心
- Email 草稿 AI 生成
- 廠商往來信件範本庫
- LINE / Email 重要訊息整理
- 廠商聯絡資訊管理

### 4. 💡 行銷發想工具
- 節慶 / 檔期行銷企劃發想
- 新產品企劃書生成
- 提案大綱快速生成
- 活動主題腦力激盪

### 5. 🧾 單據與行程
- 油資 / 高鐵票報帳整理
- 出差行程管理
- 單據拍照自動歸類
- 費用報表生成

## 語音筆記系統（跨模組基礎功能）
- 隨時語音錄入想法
- 會議逐字稿 → 摘要
- 自動歸類到對應週/月資料夾
- 定時整理推送（Telegram）

---

## 核心工作流程（報告生成）

```
平日語音/截圖/數據 → 自動歸類素材夾
        ↓
  週末一鍵召喚初稿
        ↓
AI 抓取素材 + 歷史格式 + 雲端數據
        ↓
  人工確認微調（5 分鐘）
        ↓
    輸出 PDF / Word / PPT
```

## 技術架構規劃

| 層級 | 選擇 | 說明 |
|------|------|------|
| 前端 | PWA（React） | 手機加主畫面 = 像原生 App |
| AI 核心 | Claude API / OpenAI API | 報告生成、語音轉文字 |
| 雲端同步 | Google Drive / Dropbox | 素材資料夾存放 |
| 推送通知 | Telegram Bot | 早報、提醒 |
| 報帳 OCR | Google Vision API | 發票照片辨識 |

## 商業化規劃（若測試成功）

- **第一階段**：PWA 給老婆用 → 邀 10–20 位同類職場友人測試
- **第二階段**：React Native / Flutter 包成原生 App，上架 iOS + Android
- **商業模式**：
  - 免費版：基本語音記錄、簡單報告
  - 訂閱制：$149–299/月，進階 AI 功能、雲端同步
  - 企業版：按人頭，賣給公司（高毛利）
- **注意事項**：Apple 抽成 30%（訂閱第二年 15%）、API 成本須計入毛利
